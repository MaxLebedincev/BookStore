# OnlineStore

> Онлайн магазин клиентская часть

### Включает в себя:

- components
	- CTag
	- CustomAppBar
	- CustomAuthorization
	- CustomCard
- hooks
	- access
		- useAuthorization
		- useLogout
		- useRegistration
	- data - методы работы с данными
		- add 	 - методы на получение
		- delete - методы на удаление
		- get	 - методы на получение
		- update - методы на обновление
- router
- store
- utils
	- setAuthHeader
- views
	- AboutView
	- AdminView
	- MainView
	- PersonalView
	- ProductView
